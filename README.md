# Disney Movies Data Analysis

Exploratory data analysis of Disney movies using Python, examining trends in gross revenue, genres, and release patterns.

## What This Project Does
- Analyses Disney movie revenue trends over time
- Explores relationships between genre, release date, and box office performance
- Visualises patterns and insights from the dataset

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
Disney movies dataset containing information on movie titles, release dates, genres, and total gross revenue.

## How to Run
1. Install dependencies: `pip install pandas numpy matplotlib`
2. Open `disney_movies_total_gross.ipynb` in Jupyter Notebook
3. Run all cells
